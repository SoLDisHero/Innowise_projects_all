This repository documents a set of projects through the Innowise internship program. It is structured as a series of modular tasks, each focusing on a specific area of the Data Engineering:

Core Data Processing: Manipulating complex datasets using Python, NumPy, and Pandas.

Storage & Querying: Designing schemas and optimizing performance in SQL and various Relational Databases.

Big Data Scaling: Distributed computing and data processing using Apache Spark.

Containerization & DevOps: Packaging environments and ensuring portability with Docker.

Workflow Orchestration: Building and monitoring automated data pipelines (DAGs) using Apache Airflow.

Cloud Data Warehousing: Managing enterprise-grade data storage and transformations within Snowflake.

Data Visualization: Transforming raw data into actionable insights using various BI Tools.
