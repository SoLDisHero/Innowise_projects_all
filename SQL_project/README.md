# SQL Project

## Overview

This project contains SQL queries written for the Pagila sample PostgreSQL database.

The goal of the project is to practice writing analytical and relational SQL queries using real-world structured data.

## Objectives

- Practice SQL query writing
- Work with relational database structures
- Perform data analysis using PostgreSQL
- Improve understanding of joins, aggregations, filtering, and sorting

## Database

- PostgreSQL
- Pagila sample database

## Topics Covered

- SELECT statements
- JOIN operations
- GROUP BY and aggregations
- Filtering with WHERE and HAVING
- Subqueries
- Sorting and limiting results
- Working with relational data
